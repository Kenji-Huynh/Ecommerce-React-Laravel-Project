# Thư mục Images

## Hướng dẫn sử dụng

Đặt các file hình ảnh vào thư mục này:

### Banner Images
- `banner-hero.jpg` - Hình banner chính cho trang home (kích thước khuyến nghị: 1920x800px)
- `banner-hero-mobile.jpg` - Hình banner cho mobile (kích thước khuyến nghị: 768x600px)

### Logo
- `logo.png` - Logo Pure Wear (kích thước khuyến nghị: 200x60px)

### Product Images
- Đặt hình ảnh sản phẩm trong thư mục con `products/`

### Icons
- Đặt các icon trong thư mục con `icons/`

## Gợi ý tìm hình
- Unsplash.com - Hình ảnh miễn phí chất lượng cao
- Freepik.com - Vector và hình ảnh
- Pixabay.com - Hình ảnh miễn phí

## Lưu ý
- Nên tối ưu hóa kích thước file để trang web load nhanh
- Format khuyến nghị: JPG cho ảnh, PNG cho logo có nền trong suốt
